# Group Policy Manager

An Electron.js desktop application for managing Windows Group Policy settings with a modern UI.

## Features

- Control external drive write access
- Manage browser website access policies
- Domain whitelist management
- Password-protected access with multiple recovery methods

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

## Build

```bash
npm run build
```

## Requirements

- Windows 10 or later
- Administrator privileges
- Node.js 16 or later

## Security

This application requires administrator privileges to modify Windows Group Policy settings.
